# Taxi-Fare-Prediction
Situation: Predicting taxi fares accurately requires processing complex, multi-variable datasets with high variance.

Task: Build a robust predictive model using the Chicago Taxi dataset.

Action: I have implemented a Linear Regression model using TensorFlow and Keras, performing extensive data preprocessing with Pandas and hyperparameter tuning.

Result: Successfully optimized model convergence, significantly reducing the Root Mean Square Error (RMSE) for fare estimations.
